# Challenge A1: OCI-Image mit Docker erstellen & in Docker Hub bereitstellen

---

## 1. Vorbereitung auf EC2

```bash
# Repository klonen
git clone https://gitlab.com/jannneururer/container-bootstrap.git
cd container-bootstrap/01_container
```

---

## 2. Image anpassen (Bedingungen erfüllen)

### Dateianpassungen

| Datei       | Änderung                              | Status |
|-------------|---------------------------------------|--------|
| `app.js`    | Port auf `8091` ändern                | ✅     |
| `main.css`  | Hintergrundfarbe auf `#ebd63d` (Goldgelb) | ✅ |
| `image.png` | Ersetzt mit „Modul 169"-Bild          | ✅     |

---

## 3. Docker-Image bauen

```bash
# Image mit korrektem Namen bauen
docker build -t jannneururer/webapp_jann08_8091:1.0 .
```

![docker build successful](/images/docker-build-succes.png)

---

## 4. In Docker Hub pushen

```bash
# Bei Docker Hub anmelden
docker login
# Username: jannneururer
# Password: ********

# Image pushen
docker push jannneururer/webapp_jann08_8091:1.0
```

![docker push to Docker Hub](/images/docker-push-succes.png)

---

## 5. Container starten

```bash
# Lokales Image löschen (Test, ob's von Hub kommt)
docker rmi jannneururer/webapp_jann08_8091:1.0

# Container starten (pulled automatisch von Docker Hub)
docker run -d -p 8091:8091 --name webapp-challenge jannneururer/webapp_jann08_8091:1.0

# Prüfen
docker ps
```

![docker ps zeigt laufenden Container](/images/docker-ps.png)
