# 4️ Docker Funktionstest -- hello-world

``` bash
docker run hello-world
```

Ergebnis:

    Hello from Docker!
    This message shows that your installation appears to be working correctly.

### Ablauf im Hintergrund

1.  Docker Client kontaktiert den Docker Daemon\n2.  Image wird von DockerHub geladen\n3.  Container wird erstellt\n4.  Container führt Testprogramm aus\n5.  Output wird ins Terminal gestreamt

Docker funktioniert einwandfrei.

# 6️ NGINX Webserver Container starten

``` bash
docker run -d -p 8080:80 --name web nginx
```

### Parameter

-   -d → detached (läuft im Hintergrund)\n-   -p 8080:80 → Port 8080 (Host) → Port 80 (Container)\n-   --name web → Container Name\n-   nginx → Image

Container prüfen:

``` bash
docker ps
```
